## [简介](https://github.com/SoCXin/STM32F103/wiki)

[STM32F103](https://github.com/SoCXin/STM32F103) 采用Cortex-M3内核，CPU最高速度达72 MHz。该产品系列具有16KB ~ 1MB Flash、多种控制外设、USB全速接口和CAN。
